## Description

This is a technical test for Conexa company where the requirements are to create a REST API using NestJS about handling users, authentication, authorization and handling movies from Star Wars API.

Reference to the link below for more information ⬇️

[Conexa Technical Test](https://conexatech.notion.site/Backend-Nest-Ssr-Test-d3fa3d4c614249b2afeac5d8cf98784f)

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# test coverage
$ pnpm run test:cov
```
